## 📱 **4. Адаптивность**

### 📐 **Подход к адаптивности:**
```
🖥️ Десктоп (1326px+) → 📱 Мобильные (до 666px)
```

### 🛠️ **Техники адаптивности:**
1. **Медиа-запросы** - `@media (max-width: 666px)`
2. **Display切换** - `display: none` / `display: block`
3. **Flexbox перестройка** - изменение направлений
4. **Разные изображения** - mob-* версии

### ⚠️ **Проблемные места:**
- 📱 **Не mobile-first** - стили для десктопа переопределяются
- 🖼️ **Дублирование изображений** - две копии каждого изображения
- 📏 **Жёсткие размеры** - фиксированные высоты/ширины
